### 1、注册

#### 简要描述：

```bash
用户注册接口
```

#### 请求URL：

```bash
/api/users/signup
```

#### 请求方式：

> POST

#### 参数类型

|参数|是否必选|类型|说明|
|:---------|:------:|:--------|:--------|
|name      |Y       |string   | 用户名称 |
|pswd      |Y       |string   | 用户密码 |

#### 返回示例

```bash
{
    groupid: 2,
    success: "添加用户成功"
}
```

#### 返回参数说明

|参数|类型|说明|
|:---------|:------:|:--------|
|groupid   |int     | 用户组id，1：管理员；2：普通用户 |
